# go-folder-structure
Go Folder Structure

## Clone
```
git clone git@github.com:pansachin/go-folder-structure.git
```

## Run Restfull
``` 
    cd handlers/restfull/cmd/server
    go run main.go
```

## Run Local
```
    cd handlers/local/cmd/server
    go run main.go
```

## Run PubSub
```
    cd handlers/pubsub/cmd/server
    go run main.go
```
